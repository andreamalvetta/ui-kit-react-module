# UI Kit (React Module)
Custom made node module for [React](https://reactjs.org/) projects.

## Prerequisites
- Node.js - [https://nodejs.org/en/](https://nodejs.org/en/)

## Getting started
- Clone this repo
- Run `npm install` or `yarn install`
- Run `npm link`
- To make a production build use `npm run build` or `yarn build`
